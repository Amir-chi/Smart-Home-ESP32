
# ESP32-CAM Smart Home Dashboard

🏆 **University IoT Course Final Project**

A professional, modern web dashboard for controlling an ESP32-CAM based smart home system with natural language commands, virtual lamp control, and camera integration.

---
## ✨ Features

### Mandatory Features (All Implemented ✅)

1. **Natural Language Command Interface**
   - Text input for commands
   - AI-powered command parsing
   - Real-time command processing

2. **Virtual Lamp Control**
   - 3 independent virtual lamps
   - Visual ON/OFF states with glow effects
   - Click to toggle functionality
   - Smooth animations

3. **ESP32-CAM Integration**
   - Camera flash control
   - Photo capture functionality
   - Image display with loading states
   - Skeleton loaders for better UX

4. **Modern Dashboard UI**
   - Card-based responsive layout
   - Dark/Light mode toggle
   - Glassmorphism design
   - Professional animations

5. **Real-time Status Monitoring**
   - Connection status indicator
   - Live clock & date display
   - Toast notifications
   - Visual feedback system

### Bonus Features (Implemented 🎉)

- ✅ Fully responsive (Mobile, Tablet, Desktop)
- ✅ Advanced animations & transitions
- ✅ Skeleton loading states
- ✅ Toast notification system
- ✅ Command suggestion chips
- ✅ Hover effects on all interactive elements
- ✅ Animated background
- ✅ Professional typography (Inter font)
- ✅ Icon system (Font Awesome)
- ✅ Production-ready code structure
- ✅ Clean separation of concerns
